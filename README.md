API testing
=================


API implementation with NodeJs, Express and MongoDb, deployed to Vercel.

URL: https://agile-hamlet-32644.herokuapp.com/api

Endpoints:

* GET `/api/users`
* POST `/api/users`
* GET `/api/users/:id`
* PUT `/api/users/:id`
* PATCH `/api/users/:id`
* DELETE `/api/users/:id`
