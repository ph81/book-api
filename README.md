API testing
=================

Simple book API developed with Node, Express, Mongodb and Vercel.

URL: https://book-api-black.vercel.app/api

Endpoints:

* GET `/api/books`
* POST `/api/books`
* GET `/api/books/:bookId`
* PUT `/api/books/:bookId`
* PATCH `/api/books/:bookId`
* DELETE `/api/books/:bookId`
