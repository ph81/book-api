API testing
=================


An app testing a simple API implementation with NodeJs, Express and MongoDb, with a React frontend.

URL: https://rainbow-dancing-keyboard.glitch.me/api

Endpoints:

* GET `/api/users`
* POST `/api/users`
* GET `/api/users/:id`
* PUT `/api/users/:id`
* PATCH `/api/users/:id`
* DELETE `/api/users/:id`
